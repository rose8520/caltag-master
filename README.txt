This folder allows user to obtain the position of the camera using the Caltag-frame as a reference. 


1. Run the code localizing.m.

2. Making use of the Kinect_images be sure that the Kinect is capturing and saving the images as both RGB
and DEPTH.

3. Using the Depth to find the x-y position of the camera.(playgroundepth)

4. The camera yaw, roll and pitch angle can be determined by the position of the camera.
